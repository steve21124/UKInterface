UKINTERFACE
-----------

A little study on how one could build a user interface with pretty much automatic layout from information in a user-readable XML file. I.e. this is a reimplementation of NIBs

Look at the TestUI.uki file for an example of how the UI elements would be specified.


IMPORTANT
----------

You *must* check out UliKit into a directory next to the UKInterface directory, otherwise you will be missing files this project needs.
